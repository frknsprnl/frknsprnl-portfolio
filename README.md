## üìù Customization

### Update Personal Information

1. **Hero Section**: Edit `components/Hero.tsx`
2. **About Section**: Edit `components/About.tsx`
3. **Experience**: Update `components/Experience.tsx` with your work history
4. **Projects**: Modify `components/Projects.tsx` with your projects
5. **Contact**: Update email and social links in `components/Contact.tsx`

### Styling

- Colors are defined in `tailwind.config.ts`
- Global styles in `app/globals.css`
- Component-specific styles use Tailwind utility classes

### Assets

Add your images to the `public/` folder:
- Project screenshots
- Profile picture (if needed)
- Logo/favicon
